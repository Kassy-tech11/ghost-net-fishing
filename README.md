# ghost-net-fishing
Prototypische Webanwendung zum Melden und Bergen von Geisternetzen.

## Technologien
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf

## Sprint 1 Anforderungen
1. Geisternetze anonym erfassen
2. Für Bergung eines Geisternetzes eintragen
3. Noch zu bergende Geisternetze anzeigen
4. Geisternetze als geborgen melden
5. Geisternetze als verschollen melden
